## ✨ Key Features

-   **Cinematic Hero Page:** A stunning, animated 3D landing page to welcome users.
-   **Full CRUD Functionality:** Create, Read, Update, and Delete journal entries seamlessly.
-   **Rich Journal Editor:** Includes title, content, mood tracking, and a tagging system.
-   **Comprehensive Dashboard:** At-a-glance stats including total entries, word count, writing streak, and top mood.
-   **Advanced Filtering & Sorting:**
    -   Real-time search across all entries.
    -   Filter by 8 different moods.
    -   Filter by 10 preset tags.
    -   Sort by newest, oldest, longest, or shortest.
-   **Visualizations:**
    -   **Mood Distribution Chart:** A bar chart showing the frequency of different moods.
    -   **Streak Calendar:** A GitHub-style heatmap visualizing your writing consistency over the last 28 days.
-   **Dual Theme System:** A beautiful and persistent **Light & Dark Mode**, instantly switchable.
-   **Mobile-First Responsive Design:** A flawless experience on all devices, from desktops to phones, featuring a dedicated mobile navigation bar.
-   **Polished UI/UX:** Built with a focus on micro-interactions, smooth animations, and a professional, launch-ready feel.

## 🚀 Tech Stack

| Category      | Technology                                                                                                                                                                    |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Frontend**  | **Next.js 14+** (App Router), **React 18** (Hooks), **Tailwind CSS v4**                                                                                                         |
| **Backend**   | **Supabase** (PostgreSQL Database)                                                                                                                                            |
| **Styling**   | Pure CSS with variables for theming, CSS Keyframe animations, 3D SVG animations                                                                                               |
| **Deployment**| **Vercel** / **Netlify** (CI/CD), **GitHub** (Version Control)                                                                                                                  |
